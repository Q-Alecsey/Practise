Программа написана с помощью данных средств : HTML5 / CSS / JAVASCRIPT / CHART.JS (ГРАФИКА)

Особенности:

1) При чтении / обработки / обращении к большому файлу есть небольшая задержка , зависит от веса самого файла
2) Задавать временной диапазон исключительно как написано на сайте для корректной работы 
3) Ограничение по функциям на графике : 5 штук
4) Чекбоксы не ограничивают работу друг друга
5) Столбчатый график работает медленнее линейного
6) Осреднение графиков не отобразится , если заданный временной интервал будет меньше осредненного
7) При большом количестве данных на графике , они не все отображены на осях , т.к. просто не вмещаются , однако можно навестись на сам график (или же на точку на графике), где будут отображены все данные
